# 艾宾浩斯记忆曲线背诵表

基于艾宾浩斯遗忘曲线的交互式背诵计划表，单文件 HTML，无需构建工具，浏览器打开即用。

## 功能

- **背诵计划生成** — 配置起始日期、每日项数、总天数，自动生成符合遗忘曲线复习规律的背诵计划
- **复习周期** — 遵循 1、2、4、7、15、30 天七个复习节点，科学对抗遗忘
- **连续打卡** — 追踪连续完成天数，朱红色徽章 + 脉冲动画激励
- **今日焦点视图** — 一键聚焦今日任务，Scholar's Daily Folio 概览卡片清晰展示新学内容与待复习项
- **印章庆祝动画** — 当天全部复习完成时触发盖章动画 + 烟花 + 音效
- **内容编辑面板** — 为每条背诵内容添加自定义文本，自动保存
- **浏览器通知提醒** — 支持每日定时提醒（需授权通知权限）
- **数据导出/导入** — JSON 格式导出导入，方便跨设备迁移数据
- **本地音频播放器** — 内置白噪音（雨声/白噪音/篝火）和本地音乐播放
- **打印友好** — 专门的打印样式，支持纸质输出

## 使用方法

### 直接打开

用浏览器打开 `index.html` 即可：

```
start "" "index.html"
```

或运行 `shortcut.vbs` 在桌面创建快捷方式。

### 本地服务器（推荐）

直接打开 `file://` 协议可能导致缓存问题，推荐启动本地 HTTP 服务器：

```bash
cd 项目目录
python -m http.server 8765
# 浏览器打开 http://localhost:8765/index.html
```

## 技术栈

- 纯 HTML + CSS + JavaScript，零依赖零构建
- localStorage 持久化存储
- Google Fonts（Playfair Display / Crimson Text / JetBrains Mono / Ma Shan Zheng）
- Web Audio API（盖章音效）
- CSS 自定义属性色彩系统
- IndexedDB（本地音频文件存储）

## 设计风格

复古科学笔记美学 — 羊皮纸纹理背景、铁胆墨水色系、黄铜/朱红/瓶绿色调、水彩晕染复习列、火漆印章复选框、温度计进度条。
